# UI testing framework, based on Cypress

## Configuration

All used configurateon you can find in cypress.json
{
"baseUrl": "https://www.scentbird.com",
}

## Installation

1. Install Node.js (version 12+)
2. Install dependencies

```bash
npm install
```

## Tests

- Open

```bash
npm run cypress:open
```

- Run

```bash
npm run cypress:run
```
